Git

  • git init - Создать пустой репозиторий Git или вновь инициализировать существующий.
  • git add - Добавить файлы в область подготовленных файлов.
  • git status -Просмотреть статус нужного репозитория.
  • git commit -m "Сообщение" - Создать коммит с описанием.
  • git log - Отобразить историю коммитов.
