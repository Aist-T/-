Git

  • git init -Создать пустой репозиторий Git или вновь инициализировать существующий.
  • git add -Добавить отдельный файл в область подготовленных файлов.
  • git status -Просмотреть статус нужного репозитория.
  • git commit -m -Добавить комит с описанием после знака -m.
  • git log - Посмотреть изменения внесенные в репозиторий.
